● Apply a security mindset while remaining ethical.

A. A security mindset is the ability to be able to look for and identify potential or actual compromise, that should reduce risks to assets. Modelling security policy and implement to whole system. Using security mechanisms.
-Use Access Control Lists, Firewalls, Virtual machines, using only strong passwords, limit permissions granted, Encrypt data everywhere possible, implement user activity monitoring, patching system and use latest updates, automating system security, educate your users about security policies

B. Five points of cyber ethics are:

  • 1.Your computer or system should not be used to harm others
  • 2.Your cyber knowledge should not be used to steal other people`s resources
  • 3.One should not use or copy software for which you have not paid
  • 4.You should not break into someone else`s accounts
  • 5.Never use other people`s resources without their consent
